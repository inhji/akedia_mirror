# Akedia

> „Der Dämon der Trägheit, der auch Mittagsdämon genannt wird, ist belastender als alle anderen Dämonen.“

# Features

* Topics
* Likes
* Bookmarks
* Posts (+ Images)
* Sending Webmentions
* Microformats
* Weather
* 2FactorAuth (U2F + TOTP Fallback)
* Admin Dashboard
